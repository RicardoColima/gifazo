# Gifazo

Una aplicación web para buscar y descubrir GIFs increíbles. Construida con Vue 3 y Tailwind CSS.

## Características

- 🔍 Búsqueda de GIFs en tiempo real
- 📱 Diseño responsive y moderno
- 🎨 Animaciones y efectos visuales atractivos
- 📜 Scroll infinito para explorar más contenido
- 🎲 Botón "Sorpréndeme" para descubrir GIFs aleatorios

## Tecnologías

- Vue 3
- Tailwind CSS
- Tenor API
- Vite

## Instalación

```sh
npm install
```

## Desarrollo

```sh
npm run dev
```

La aplicación se ejecutará en: http://localhost:5173/gifazo/

## Producción

```sh
npm run build
```

## Nota importante

Este proyecto está configurado para funcionar con GitHub Pages. El `base path` está configurado como `/gifazo/`.

### Para desarrollo local:
- Accede a: http://localhost:5173/gifazo/
- No accedas a: http://localhost:5173/ (no funcionará correctamente)

### Para clonar en otro equipo:

```sh
git clone https://github.com/RicardoColima/gifazo.git
cd gifazo
npm install
npm run dev
```

Luego abre http://localhost:5173/gifazo/ en tu navegador.
